========================================================================
       MICROSOFT FOUNDATION CLASS ライブラリ : VPC
========================================================================

VPC アプリケーションは AppWizard が生成しました。
このアプリケーションは、これから作るアプリケーションのひな型であるだけでなく、
Microsoft Foundation Class の基本的な使い方を説明しています。

このファイルには VPC アプリケーションを構成している各ファイルの概要説明が
含まれています。


VPC.MAK
    このプロジェクト ファイルは Visual C++ Workbench 用メイクファイルと互換性
    があります。

    また、プロフェッショナル版 Visual C++ に用意されている NMAKE プログラム用
    メイクファイルとも互換性があります。

    MS-DOS プロンプトからデバッグ用のプログラムをビルドするには、
      nmake DEBUG=1 /f VPC.MAK
    と入力してください。

    また、リリース用のプログラムをビルドするには、
      nmake DEBUG=0 /f VPC.MAK
    と入力してください。

VPC.H
    このファイルはアプリケーションの中心となるインクルードファイルです。この
    ファイルは他のプロジェクトの固有のインクルードファイル（RESOURCE.H  も含
    みます） をインクルードし、また CVpcApp アプリケーションクラスを宣
    言します。

VPC.CPP
    このファイルは CVpcApp アプリケーションクラスを含むアプリケーション
    の中心となるソースファイルです。

VPC.RC
    このファイルには、アプリケーションで使用するすべての Microsoft Windows
    用リソース一覧が記述されています。このファイルは RES サブディレクトリに
    格納されているアイコン、ビットマップ、カーソルなどをインクルードしていま
    す。このファイルは App Studio で直接編集できます。

RES\VPC.ICO
    このファイルは、アプリケーションのアイコンとして使用されるアイコンファイ
    ルです。このアイコンはリソース ファイル VPC.RC によってインクルード
    されます。

RES\VPC.RC2
    このファイルには App Studio で編集できないリソースを記述します。生成時に
    はアプリケーション用にカスタマイズ可能な VERSIONINFO  リソースを持ってい
    ます。 App Studio で編集できないリソースは、このファイルに記述してくださ
    い。

VPC.DEF
    このファイルはアプリケーションが Microsoft Windows 上で実行される時に必
    要な情報を持っています。例えばアプリケーションの名前や説明、ローカルヒー
    プの初期値などのパラメータを定義します。このファイルの中のパラメータの数
    値は Microsoft Foundation Class ライブラリを用いたアプリケーションにおけ
    る標準的なものです。デフォルトのスタック サイズはプロジェクト ファイルを
    編集して調整します。

VPC.CLW
    このファイルは ClassWizard  が既存のクラスの編集やクラスを新規作成すると
    きに使う情報を持っています。またこのファイルは、ClassWizard によるメッセ
    ージ マップやダイアログ データ マップの編集や生成、プロトタイプ メンバ関
    数の生成に必要な情報も持っています。

/////////////////////////////////////////////////////////////////////////////

メイン フレーム ウィンドウ：

MAINFRM.H, MAINFRM.CPP
    These files contain the frame class CMainFrame, which is derived from
    これらのファイルはフレーム クラス CMainFrame を含みます。
    フレーム クラス CMainFrame は CFrameWnd クラスから派生し、SDI の
    フレームを制御します。


/////////////////////////////////////////////////////////////////////////////

AppWizard は一つのドキュメントと一つのビューを作成します。

VPCDOC.H, VPCDOC.CPP - ドキュメント
    これらのファイルは CVpcDoc クラスを含みます。特別なドキュメント デ
    ータを付け加えたり、CVpcDoc::Serialize を使ったファイルのセーブや
    ロード機能をインプリメントしたいときはこのファイルを編集してください。

VPCVIEW.H, VPCVIEW.CPP - ドキュメントのビュー
    これらのファイルは CVpcView クラスを含みます。CVpcView  オブ
    ジェクトは CVpcDoc オブジェクトをビューするために使います。



/////////////////////////////////////////////////////////////////////////////
その他の標準ファイル：

STDAFX.H, STDAFX.CPP
    これらのファイルは STDAFX.PCH という名前のプリコンパイル済ヘッダー  ファ
    イル（PCH) や STDAFX.OBJ という名前のプリコンパイル済型ファイル (PCT) を
    構築するために使うファイルです。

RESOURCE.H
    このファイルは新規のリソース ID を定義するための標準のヘッダー ファイルです。
    App Studio はこのファイルを読んだり更新したりします。
